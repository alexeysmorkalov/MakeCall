MakeCall - модуль выполненный в виде консольного приложения, 
который подключается через TAPI к мини атс АГАТ ux 5110
и выполняет набор номера с заданного внутреннего номера SIP АТС.
============================================================

* Требования
Программа требует установленного .Net Framework 4.0
http://www.microsoft.com/ru-ru/download/details.aspx?id=17718

Программа требует, чтобы на станции был установлен и настроен TSP-драйвер TAPI 

* Установка 
Программа не требует установки

* Состав
MakeCall.exe
MakeCall.exe.Config - файл конфигурации
Atapi.dll - упраляемая библиотека доступа к TAPI http://atapi.codeplex.com/

* Конфигурация
В файле конфигурации MakeCall.exe.Config задаются 2 параметра - 
входная и выходная папка заданий.
Важный момент - в конфигураторе АТС для всех внутренних (трехзначных) номеров абонентов
в параметре "Режим работы функции TapiMakeCall" должно стоять значение "Обратный вызов".

* Запуск и останов.
Запуск любым способом MakeCall.exe
Останов - клавишей ESC. Рекомендуется делать именно так, а не через Ctrl+C, 
т.к. корректно финализируется TAPI
